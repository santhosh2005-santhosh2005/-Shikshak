**DysCover** is an intelligent learning assessment and screening platform designed to identify learning patterns and provide actionable insights for personalized education. By combining cognitive science principles with modern web technology, DysCover provides a comprehensive understanding of unique learning profiles.

---

## 🎯 Our Mission
Learning differences affect many students, yet early identification remains challenging. DysCover's mission is to:
- **Democratize Access**: Provide high-quality assessment tools without barriers.
- **Provide Clarity**: Deliver actionable insights and recommendations for parents and educators.
- **Empower Learning**: Help every learner succeed with personalized strategies.

---

## ✨ Key Features

### 🔍 Comprehensive Assessment Suites
Tailored assessments categorized by age groups to ensure age-appropriate cognitive load:
- **Phonological Awareness**: Identification of phonemes and sound-letter relationships.
- **Working Memory**: Visual and auditory sequencing tasks to measure recall capacity.
- **Rapid Naming & Recognition**: Measuring speed of lexical access.
- **Reading & Orthography**: Analyzing spelling patterns and reading fluency.

### 📊 Intelligent Insights Dashboard
Advanced analytics that explain WHY and WHAT TO DO:
- **Smart Analysis**: AI-powered insights that explain learning patterns.
- **Strength Mapping**: Highlights where the learner excels.
- **Personalized Recommendations**: Actionable steps and teaching strategies.
- **Learning Style Identification**: Understand how each learner processes information best.

### ♿ Accessibility-First Design
Built from the ground up for all learners:
- **OpenDyslexic Support**: One-click toggle for specialized fonts.
- **Text-to-Speech (TTS)**: Integrated verbal guidance for all assessments.
- **Adjustable Spacing**: Control over line and letter spacing.
- **High-Contrast Modes**: Color palettes optimized for visual comfort.

## 🛠️ Technical Architecture

### Core Stack
- **Frontend**: React 18 with TypeScript for type-safe state management.
- **UI/UX**: Modern component library built on Radix UI primitives.
- **State & Transitions**: `framer-motion` for smooth animations.
- **Backend**: `Supabase` for secure data management.
- **Analytics Engine**: Custom insight generation system for intelligent analysis.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/)

### Installation
1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd dyscover
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=https://your-project.supabase.co
   VITE_SUPABASE_ANON_KEY=your-anon-key
   ```

4. **Launch Development Server:**
   ```bash
   npm run dev
   ```
